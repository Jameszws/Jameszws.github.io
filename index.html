@tc-i18n前端国际化解决方案

tc-i18n是一个轻量级、易集成的多语言库，支持多技术栈和多端应用能力。特点是业务无侵入、框架无关性，可以让开发者轻松实现多语言支持，无需改动现有业务代码逻辑，适用于Web、移动等平台。

什么是 @tc-i18n ？

Github

易用性

2分钟快速接入，同时支持npm包和CDN形式接入，您的项目即可无缝升级为多语言版本，无需繁琐地定义众多语言键值对，也无需对原始代码进行任何修改。

🧰

多技术栈支持

目前已经全面支持Vue、React以及微信小程序等主流开发框架，并且通过插件机制，我们能够进一步扩展对更多框架的支持，以满足您的多样化开发需求。

🛠️

扩展性

提供强大的插件扩展功能，包括自定义钩子和灵活的插件定制选项，以满足项目个性化需求。

📚

业务无侵入

只需轻松安装npm包，您的项目即可无缝升级为多语言版本，无需繁琐地定义众多语言键值对，也无需对原始代码进行任何修改。

📝️

多端支撑

支持 Web端、APP端、桌面端、小程序端 等等。
